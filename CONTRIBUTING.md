# Contributing

Thanks for helping improve OpenAgents Kit.

## How To Contribute

- Keep changes focused and reviewable.
- Update docs when behavior changes.
- Add or adjust tests for meaningful code changes.
- Keep the scaffold portable and tool-agnostic.

## Before You Open A PR

- Run the project checks relevant to your change.
- Make sure generated outputs stay deterministic.
- Avoid hard-coded personal paths or account-specific assumptions.

## Scope

This repo is a reusable template, so changes should improve the scaffold itself or a documented extension point.
