# Security Policy

If you find a security issue in OpenAgents Kit, please report it privately instead of opening a public issue.

## What To Include

- A clear description of the problem
- The affected version or commit
- Reproduction steps, if available
- Any relevant logs or screenshots

## What Not To Include

- Secrets
- API keys
- Tokens
- Production credentials

## Response

We will review reports as quickly as possible and coordinate a safe fix or disclosure path.
